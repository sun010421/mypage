1. 텍스트 수정 (화면에서 바로 가능)

업무 항목명과 마일스톤 텍스트를 클릭하면 인라인 편집됩니다. 다른 곳 클릭 시 반영됩니다. (새로고침하면 원본으로 돌아갑니다)

2. 바(일정) 수정 — data 배열 편집

HTML 파일을 VS Code 등 에디터로 열고 const data = [...] 배열에서 각 task의 start/end 값을 변경하세요.

// 예: "TCAD 전계 해석" 3월~6월 → 3월~8월로 변경 { type: 'task', label: 'TCAD 전계 해석 및 구조 설계', track: 't1', start: 0, // 0=3월(시작월) end: 5, // 3→5로 변경 (6월→8월) tip: '툴팁 설명' }

3. 업무 항목 추가/삭제

// 새 업무 추가 (track: 't1'/'t2'/'t3') { type: 'task', label: '새 업무', track: 't2', start: 2, end: 6, tip: '설명' } // 새 마일스톤 추가 { type: 'milestone', label: '중간 검토', month: 5 } // 삭제: 해당 객체 줄을 지우면 됨
